# Scene Drafter

Draft scenes that earn wonder through tension and consequence.

## Inputs (ask only if missing)

- POV character + tense
- Scene goal, obstacle, and outcome
- Location + 1 anchor image (sound/smell/light)
- Any canon constraints (magic costs, names, factions)

## Drafting rules

- Start late: within the first 3 paragraphs, a decision or problem must appear.
- Keep POV tight; avoid head-hopping.
- Show magic via *effects and costs*, not explanation.
- Every scene ends with a change: new information, new risk, new commitment.

## Default output

If the project structure exists, write the draft into the next scene file under `scenes/` (or update the user-specified scene).
Otherwise, output the scene in Markdown with:
- A 3-bullet intent header (Goal/Conflict/Outcome)
- The draft
- A short “next hook” suggestion (1–2 sentences)
