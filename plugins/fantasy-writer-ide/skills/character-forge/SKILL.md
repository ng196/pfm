# Character Forge

Create characters as engines of choice under pressure.

## Default output

For each character, produce:
- Role in story + “what they *do* on the page”
- Desire vs need
- Wound, lie, and coping strategy
- Moral line (what they won’t do) and the condition that could break it
- Secret (stakes if revealed)
- Voice: rhythm, vocabulary, 3 signature phrases, 2 tells (physical/behavioral)
- Relationship web: 3 ties (ally/rival/complication)

If the project structure exists, write one file per character under `characters/` using a stable filename (for example `characters/elin-ravane.md`).

## Checks

- Every major character must want something that clashes with someone else’s want.
- Give each character a “competence” and a “blind spot” that causes plot.
